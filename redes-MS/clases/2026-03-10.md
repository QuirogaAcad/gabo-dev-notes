## 🖥 Comandos de Red

### `ping`

Comprueba si un dispositivo es alcanzable en la red.

```bash
ping -t 192.168.1.1
```

| Parte | Descripción |
|-------|-------------|
| `ping` | Envía paquetes ICMP |
| `-t` | Envía paquetes continuamente |
| `192.168.1.1` | Dirección IP del router local |

---

### `tracert`

Muestra la ruta que siguen los paquetes hasta un destino.

```bash
tracert 8.8.8.8
```

| Parte | Descripción |
|-------|-------------|
| `8.8.8.8` | Servidor DNS público de Google |
| hops | Muestra los saltos entre routers |

---

### `ipconfig /flushdns`

Limpia la caché DNS del sistema. Útil para resolver problemas de resolución de nombres.

```bash
ipconfig /flushdns
```

## 🐧 Herramientas en Linux

### `tcpdump`

Herramienta de análisis de tráfico de red desde la terminal.

```bash
tcpdump -i eth0 icmp
```

| Parte | Descripción |
|-------|-------------|
| `-i eth0` | Interfaz de red |
| `icmp` | Filtra paquetes ICMP (como ping) |
