# 🐧 Virtualizar e Instalar Linux

📅 **Fecha:** 19/03/2026

---

## 🧠 Concepto general

La virtualización permite ejecutar un sistema operativo dentro de otro sin afectar el sistema principal.

**Ejemplo:**
- **Host:** Windows 10
- **Invitado:** Linux (en una máquina virtual)

---

## 💻 Requisitos básicos de hardware

| Componente | Especificación |
|------------|----------------|
| CPU        | i5 o superior  |
| RAM        | 8 GB recomendados |
| Disco      | SSD (mínimo 500 GB recomendado) |
| Red        | Ethernet / WiFi |
| GPU        | Opcional (según uso) |

---

## 🖥️ Virtualización

### 🔹 VirtualBox

Permite crear máquinas virtuales dentro de tu sistema operativo.

**Estructura:**

```
Windows 10 (Host)
└── Máquina Virtual
    ├── Linux
    │   ├── CPU asignada
    │   ├── RAM asignada
    │   └── Disco virtual
```

### 🔹 Tipos de uso dentro de una VM

- Base de datos (DB)
- Servidores web
- Testing de sistemas

---

## 🧩 Alternativa: WSL

**WSL (Windows Subsystem for Linux)** permite usar Linux dentro de Windows sin máquina virtual.

**Ventajas:**
- Menor consumo de recursos
- Integración directa con Windows
- Arranque rápido

**Comandos básicos en WSL:**

```bash
pwd     # Muestra el directorio actual
cd      # Cambia de directorio
exit    # Cierra la terminal
```

**Ruta típica:**

```
/home/usuario
```

## 🖥️ Shell y Bash

### 🐚 ¿Qué es el Shell?

El Shell es un intérprete de comandos que permite interactuar con el sistema operativo mediante texto.

> 👉 Es el intermediario entre el usuario y el sistema.

### ⚙️ ¿Qué es Bash?

**Bash (Bourne Again Shell)** es uno de los shells más usados en Linux.

**Características:**
- Autocompletado
- Historial de comandos
- Scripting (automatización)

---

## 📂 Comandos básicos

### 📁 Navegación

```bash
pwd        # Muestra el directorio actual
ls         # Lista archivos y carpetas
ls -l      # Lista detallada
cd carpeta # Entra a una carpeta
cd ..      # Retrocede un nivel
```

### 📦 Gestión de archivos

```bash
mkdir nombre_carpeta   # Crear carpeta
rm archivo             # Eliminar archivo
rm -r carpeta          # Eliminar carpeta
cp origen destino      # Copiar
mv origen destino      # Mover o renombrar
```

### ⌨️ Atajos útiles

```
Ctrl + Shift + Esc  # Abrir administrador de tareas (Windows)
Windows + Espacio   # Cambiar idioma del teclado
```

## 🧠 Conceptos clave

| Término | Descripción |
|---------|-------------|
| **Host** | Sistema principal (ej: Windows) |
| **Guest** | Sistema virtualizado (ej: Linux) |
| **VM** | Máquina virtual |
| **WSL** | Linux dentro de Windows sin virtualizar completamente |

---

## 🚀 Resumen

Podés usar Linux mediante:

- **Máquina virtual** (VirtualBox) → Mayor aislamiento, más recursos
- **WSL** → Más liviano, integrado en Windows

Bash es el shell más común y aprender sus comandos básicos es clave para moverte en Linux.
