## 💻 Linux y DevOps

Linux es uno de los sistemas operativos más utilizados para servidores, desarrollo backend e infraestructura de aplicaciones.

**Ejemplo de stack:**

```
Apache  PHP  MySQL
```

---

## 🖥 Arquitectura Básica

```
Usuario → Shell → Kernel → Hardware
```

| Capa | Descripción |
|------|-------------|
| **Usuario** | Persona que interactúa con el sistema |
| **Shell** | Programa que permite ejecutar comandos |
| **Kernel** | Núcleo del SO — gestiona CPU, memoria, dispositivos y procesos |

---

## 🐚 Bash

**Bash (Bourne Again Shell)** es uno de los shells más usados en Linux. Permite ejecutar comandos, crear scripts y administrar el sistema.

---

## 📂 Todo en Linux es un Archivo

Uno de los principios fundamentales de Linux: muchos recursos del sistema se representan como archivos.

```bash
wc -l archivo.txt   # cuenta la cantidad de líneas
```

---

## 📁 Directorios Importantes

| Directorio | Descripción |
|------------|-------------|
| `/proc` | Información del sistema y procesos |
| `/dev` | Dispositivos del sistema (`/dev/sda`, `/dev/sda1`, `/dev/sdb`) |
| `/etc` | Archivos de configuración del sistema |

---

## 🔗 Pipes

El símbolo `|` permite enviar la salida de un comando como entrada de otro.

```bash
comando1 | comando2
```

---

## 🔎 Comando `grep`

Busca texto dentro de archivos.

```bash
grep palabra archivo.txt
```

---

## 🖥 Hardware Básico

Componentes mencionados: **CPU**, **RAM**, **Disco (SD)**, **Red (ETH)**

---

## 🐧 Distribuciones Linux

| Distro | Orientación |
|--------|-------------|
| **Ubuntu** | Escritorio y servidores |
| **Debian** | Servidores — muy estable |
| **Arch Linux** | Usuarios experimentados |
| **Linux Mint** | Escritorio (basada en Ubuntu) |
| **CentOS / Red Hat** | Entornos empresariales e infraestructura |

---

## 📦 Gestión de Paquetes

En sistemas basados en Debian se utiliza **APT**.

```bash
apt install mysql-server
apt install openssh-server   # permite conexión remota vía SSH
```
