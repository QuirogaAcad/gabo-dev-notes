
# 📚 Base de Datos

Recursos y herramientas para aprender sobre bases de datos relacionales y no relacionales.

## 🛠️ Software Recomendado

### SQL y SGBD Relacionales

| Herramienta | Descripción | Descarga |
|-------------|-------------|----------|
| **MySQL** | Sistema de gestión de bases de datos relacional | [mysql.com](https://www.mysql.com/) |
| **MySQL Workbench** | Interfaz gráfica oficial para MySQL | [mysql.com/workbench](https://www.mysql.com/products/workbench/) |
| **PostgreSQL** | SGBD relacional avanzado | [postgresql.org](https://www.postgresql.org/) |

### NoSQL

| Herramienta | Descripción | Descarga |
|-------------|-------------|----------|
| **MongoDB** | Base de datos de documentos | [mongodb.com](https://www.mongodb.com/) |
