## 🤖 Placas Programables

Dispositivos electrónicos que pueden ejecutar programas para controlar componentes físicos.

### Arduino

Plataforma de hardware y software libre utilizada para:

- Proyectos electrónicos

Placa educativa diseñada para enseñar programación y electrónica. Se utiliza en educación STEM para:

- Proyectos interactivos

## 🧩 Componentes Externos (Módulos)

Los módulos permiten ampliar las capacidades de las placas programables.

### Sensores
Detectan información del entorno.

Ejemplos: temperatura, luz, movimiento, distancia

### Actuadores
Realizan una acción física.

Ejemplos: motores, LEDs, relés, servomotores

### Elementos de Control
Controlan o modifican el comportamiento del circuito.

Ejemplos: botones, potenciómetros, interruptores

### Alimentación y Protección
Suministran energía y protegen los circuitos.

Ejemplos: baterías, resistencias, reguladores de voltaje

## ⚡ Electricidad Básica

### Ley de Ohm

Describe la relación entre Voltaje, Corriente y Resistencia.

| Símbolo | Magnitud | Fórmula |
|---------|----------|---------|
| **V** | Voltaje | `V = I × R` |
| **I** | Corriente | `I = V / R` |
| **R** | Resistencia | `R = V / I` |

## 🔌 Circuitos Eléctricos

### Circuito en Serie

Los componentes están conectados uno después del otro.

```
Batería ── R1 ── R2 ──
```

**Resistencia total:**
```
RT = R1 + R2
```

- La **corriente es la misma** en todos los componentes
- El **voltaje se reparte** entre las resistencias

---

### Circuito en Paralelo

Los componentes se conectan en ramas separadas.

```
         ├─ R1 ─┤
Batería  ├─ R2 ─┤
```

- El **voltaje es el mismo** en cada rama
- La **corriente se divide** entre los caminos
